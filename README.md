# 📊 Customer Churn Prediction Using Machine Learning

## 📌 Project Overview

Customer Churn Prediction is a Machine Learning project that predicts whether a customer is likely to leave a company's service based on historical customer data. The project uses the **Random Forest Classifier** to analyze customer behavior and identify customers who are at risk of churning, helping businesses improve customer retention strategies.

---

## 🎯 Objective

The main objective of this project is to build a classification model that can accurately predict customer churn using historical customer information. By identifying customers who are likely to leave, businesses can take proactive measures to retain them.

## 📊 Machine Learning Algorithm

This project uses the **Random Forest Classifier**, an ensemble learning algorithm that combines multiple decision trees to improve prediction accuracy and reduce overfitting.
